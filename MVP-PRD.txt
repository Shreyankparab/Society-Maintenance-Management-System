Society Maintenance Management System
Complete Requirement Discovery & Scope Document
1. Project Goal
A web-based system where:
Society admins/manage committee members can:
Manage flats & residents
Generate maintenance bills
Track paid/unpaid dues
Send reminders
Manage receipts
View reports
Flat owners/residents can:
Login securely
View maintenance dues
Pay online
Download receipts
View payment history
Raise complaints/requests (optional)
2. User Roles
A. Super Admin (You / Society Head)
Highest control.
Permissions
Create/manage society
Add committee members
Configure maintenance structure
View all reports
Backup/export data
Manage payment gateway settings
Manage notices/broadcasts

B. Committee/Admin Staff
Permissions
Add/edit residents
Add flats/wings
Generate maintenance invoices
Mark payments manually
Approve offline payments
Generate receipts
Send reminders
View defaulters list

C. Flat Owner / Resident
Permissions
Login
View dues
Pay maintenance
Download receipt
View notices
Update profile
View payment history
